# Pinterest v2 — Metadata CSVs

## Files in this folder

- `pinterest_upload_sheet.csv` — Paste into Pinterest when uploading pins. One row per pin.
- `pinterest_tracking_sheet.csv` — Track performance metrics after publishing. One row per pin.
- `manifest.json` — Generated by render_pinterest_v2.py. Lists all PNG files with metadata.

## How to import to Google Sheets

1. Go to [Google Sheets](https://sheets.google.com) and create a new spreadsheet.
2. Click **File → Import**.
3. Upload `pinterest_upload_sheet.csv`.
4. Select **"Replace spreadsheet"** or **"Insert new sheet"**.
5. Click **Import data**.
6. Repeat for `pinterest_tracking_sheet.csv` in a second sheet tab.

## Upload sheet columns

| Column | Description |
|---|---|
| pin_id | Unique pin identifier (pin-001 through pin-032) |
| file_name | PNG filename to upload to Pinterest |
| board_name | Target Pinterest board |
| pin_title | Pin title (shown on Pinterest) |
| pin_description | Pin description — copy/paste into Pinterest upload form |
| destination_url | Destination URL with UTM tracking |
| alt_text | Accessibility alt text for the image |
| keywords | Relevant keywords separated by ` | ` |
| style | Visual style: Coastal, Minimalist, Sunset, Classic |
| property | Property name |
| template | Template used |
| status | `ready` = PNG generated, `pending` = not yet generated |

## Tracking sheet columns

After publishing, fill in these columns weekly:

| Column | Description |
|---|---|
| pinterest_url | Full URL of the published pin |
| publish_date | Date published (YYYY-MM-DD) |
| status | pending / live / paused / deleted |
| impressions | Views from Pinterest analytics |
| saves | Pin saves (repins) |
| outbound_clicks | Clicks from Pinterest to your site |
| landing_visits | Confirmed GA4 sessions from Pinterest |
| cta_clicks | Clicks on CTA buttons on landing page |
| leads | Email signups or Tally form starts |
| purchases | Stripe payments confirmed |
| ctr | outbound_clicks / impressions |
| lead_rate | leads / outbound_clicks |
| purchase_rate | purchases / outbound_clicks |
| notes | Notes on performance, creative, etc. |

## Winner criteria (after 7–10 days)

Declare a pin a winner if it shows:
- High outbound click rate (>1% CTR)
- High saves relative to impressions
- Confirmed leads or purchases from UTM tracking
- Clear visual clarity vs. lower-performing pins
