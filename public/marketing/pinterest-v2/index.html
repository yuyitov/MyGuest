How to render pins

  • Install: pip install playwright==1.53.0 && playwright install chromium
  • Optional: python scripts/marketing/capture_pinterest_assets.py
  • Validate: python scripts/marketing/validate_pinterest_copy.py
  • Render: python scripts/marketing/render_pinterest_v2.py --base-url http://127.0.0.1:8000/marketing/pinterest-v2/templates/
  • Metadata: python scripts/marketing/build_pinterest_metadata.py
  • Output: output/pinterest-v2/pins/ (32 PNGs)

Data files

  • data/marketing/pinterest_demo_inventory.json — 4 demos
  • data/marketing/pinterest_campaign_v2.json — 32 pins
  • data/marketing/pinterest_keywords.json — SEO keywords
  • output/pinterest-v2/metadata/pinterest_upload_sheet.csv
  • output/pinterest-v2/metadata/pinterest_tracking_sheet.csv
